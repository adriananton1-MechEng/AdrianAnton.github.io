Portfolio admin

Edit any text, toggle highlights, change fonts and colors, reorder cards — all changes commit to GitHub and auto-deploy via Pages in ~30 seconds.

First-time setup — generate a token (one-off):

1. Open github.com/settings/personal-access-tokens/new

2. Under Repository access, choose Only select repositories → pick portofolio.

3. Under Repository permissions, set Contents to Read and write.

4. Click Generate token, copy it, and paste below:

The token is stored only in this browser's localStorage. It is never sent anywhere except api.github.com. You can revoke it at any time from GitHub settings.
