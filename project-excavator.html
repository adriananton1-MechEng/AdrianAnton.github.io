Overview
Eight servos, eight valves — manual lever bank, automated

A retrofit project on a mini excavator that replaces the manual hydraulic lever bank with eight servo motors driving the eight spools of the original hydraulic distributor. Each valve gets its own servo, so every function the operator could perform manually — track left, track right, swing, boom, arm, bucket, blade, auxiliary — becomes an electronic command instead of a hand on a lever.

Before any hardware was touched, the entire machine was modelled in Fusion 360 with kinematic joints so the boom-arm-bucket linkage, swing and undercarriage motion could be driven directly from valve-position inputs in CAD. That made it possible to validate reach envelopes, joint travel limits and servo torque requirements before mounting anything to the real distributor — no surprises on commissioning day.

How it works
From CAD kinematics to physical actuation
  • Fusion 360 kinematic model — full assembly with joints on boom, arm, bucket, swing and tracks. Servo angles drive valve spool position, which drives joint motion.
  • Servo-to-spool linkage — each servo is mechanically coupled to a single distributor spool via a printed/machined adaptor; original lever bank can be reinstalled if needed.
  • Reach envelope verified in CAD — boom, arm and bucket travel limits validated against the real machine geometry before any hardware was modified.
  • Servo torque budget — required spool force estimated from CAD-derived valve travel and verified against servo stall-torque specs.
  • Electronic command layer — eight servo signals replace the eight manual levers. The control input can be a joystick, a script, or a higher-level autonomy stack.
  • Coordinated motions — software can now combine valves (e.g. boom + arm + bucket) for repeated cycles that would be tedious or imprecise by hand.
  • Safe-by-design fallback — losing power returns all servos to centred (closed-spool) state, so the machine fails to a neutral hydraulic position.
  • Reversible retrofit — no irreversible modification to the distributor itself, so the original manual lever bank can be put back if required.
Why automate it
A path toward repeatable, programmable earth-moving

Manual hydraulic excavators rely entirely on operator skill for every motion — precision suffers on repeated tasks and small machines are hard to operate remotely. Driving the distributor with servos turns the whole machine into a programmable mechatronic platform: the same valves still control the same hydraulic functions, but the inputs come from electronics instead of a hand on a lever.

From there, anything downstream becomes possible — joystick teleoperation, recorded motion playback for repeated trenching cycles, sensor-driven safety limits, or higher-level autonomy. The CAD-first approach with a full kinematic simulation means each new control idea can be validated in software before it ever moves a real cylinder.
